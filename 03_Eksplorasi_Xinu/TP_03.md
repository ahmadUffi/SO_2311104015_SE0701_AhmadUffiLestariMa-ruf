## Dasar Teori

Pertanyaan-pertanyaan ini spesifik ke **lingkungan Xinu yang sedang berjalan (runtime)**, jadi jawabannya **tidak selalu sama** tergantung hasil eksekusi di sistem kamu (biasanya di lab/VM/router Xinu).

## **Jawaban + Cara Mendapatkannya di Xinu**

### **1. Berapa jumlah perintah pada Xinu?**

Jumlah perintah di Xinu umumnya sekitar:

- **± 20–30 command**

Untuk melihat jumlah pasti:

```
help
```

Lalu hitung jumlah command yang muncul.

---

### **2. Sebutkan 2 perintah yang mempunyai fungsi yang sama!**

Contoh yang sering dianggap mirip:

- `memstat` dan `free` → sama-sama menampilkan info memori
- atau
- `ps` dan `kill` (dalam konteks manajemen proses, walaupun tidak identik)

Jawaban aman:

- **memstat dan free (informasi memori)**

---

### **3. Berapa IP address Xinu?**

Biasanya (default lab Xinu):

- **192.168.1.2** _(bisa berbeda tergantung setup)_

---

### **4. Perintah apa yang digunakan untuk mengetahui IP address?**

Gunakan:

```
ifconfig
```

atau:

```
ipaddr
```

(tergantung versi Xinu)

---

### **5. Berapa IP DNS server yang digunakan oleh Xinu?**

Biasanya:

- **8.8.8.8 (Google DNS)**
  atau mengikuti konfigurasi jaringan.

Cek dengan:

```
netstat
```

atau lihat konfigurasi network.

---

### **6. Terdapat berapa proses yang sedang berjalan pada Xinu?**

Biasanya sekitar:

- **5 – 10 proses**

Untuk cek:

```
ps
```

---

### **7. Proses apa yang mempunyai prioritas paling rendah?**

Biasanya:

- **idle process**

Karena:

- selalu jalan saat CPU tidak digunakan
- prioritas paling rendah

---

### **8. Proses apa yang mempunyai ukuran paling besar?**

Dilihat dari:

```
ps
```

Biasanya:

- **main process** atau **network-related process**

---

### **9. Proses apa yang berada dalam state current?**

- Proses yang sedang dieksekusi saat ini

Cek:

```
ps
```

Biasanya:

- **main** atau **shell**

---

### **10. Proses apa yang berada dalam state suspend?**

Biasanya:

- proses yang dihentikan sementara

Cek:

```
ps
```

Contoh:

- proses yang dibuat tapi belum dijalankan penuh

---

### **11. Berapa PID (Process ID) dari Main process?**

Biasanya:

- **PID = 0 atau 1** (tergantung versi Xinu)

Cek:

```
ps
```
